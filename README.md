# ValeMove organizado

Estrutura gerada sem alterar a lógica principal do projeto:

- `index.html` → estrutura da página
- `css/styles.css` → estilos que estavam inline
- `js/tailwind-config.js` → configuração do Tailwind que estava inline
- `js/app.js` → lógica principal da aplicação
- `assets/img/` → pasta reservada para imagens e logos
- `assets/fonts/` → pasta reservada para fontes locais, se desejar usar depois

## Observação
O projeto continua usando dependências por CDN (`Tailwind`, `Lucide` e `Leaflet`), como no arquivo original.
